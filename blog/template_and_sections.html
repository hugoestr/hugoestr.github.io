Template and Sections

I needed a template. While working on the template, I reviewed semantic sections in HTML 5.

I wrote notes for a technical article that I am writing for the site. I was ready to start writing a cleaner draft, and I ran into my first problem with the minimalismtic website. The issue was that I needed to make a second page that follows the format of the first one. This wasn't that hard to solve. All what I did was to copy the previous entry and clean the content. Whenever I need to create another page, I can jump into the shell and write: cp template.html <new_page.html> That is all what I need to do to solve my current problem.

While I was making the template, I decided to review the semantic sections that were included in HTML 5 so that I can include those elements. If you recall, HTML 5 included elements such as <article> <header> <footer>, etc. You can find details about these elements in MDN's HTML Elements page. MDN has an article that describes a use case for these html 5 semantic sections
