Setting up a .net project to do TDD with NUnit

The basic steps in adding NUnit to a C# project

These are the steps for setting up a testing C# project with NUnit so that we can do automated test units.

Steps

  • Select the solution folder. Right click and add a new project.
  • Select a C# library project. Name it .Unit. So if your project name is "Calculator", your project should be called Calculator.Unit
  • Now we install NUnit to the test project. Open the nuget console. Type 
    
                Install-Package NUnit
    So, following the same example as above, you would enter 
    
                Install-Package NUnit Calculator.Unit
    Keep in mind that if you don't include the test project name, NUnit will be installed on your main library. It is not necessary there, so let's not install it there.
  • Then install the adapter. This will help you see the tests running in Visual Studio. You go to extensions, click online, and search for NUnit Adapter. Pick NUnit3 Adapter
