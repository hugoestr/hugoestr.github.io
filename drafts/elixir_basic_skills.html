Elixir Basic Skills

A basic skills checklist with Elixir answers

Print to console
IO.puts "Hello World"
Run script on the console
The file should have the extension .exs for Elixir script. 
$ elixir hello_world.exs
Read arguments from console
[first| rest] = System.argv
Create a starndard project
$ mix new <project_name>
Write to a file
From Elixir's Getting Started 

              {:ok, file} = File.open("hello", [:write])
              IO.binwrite(file, "world")
              File.close(file)
Read a file
From Elixir's Getting Started 

              File.read("file_name")
Write a function and a module
All functions must be in modules 

              defmodule Hello do
                def salutation do
                  IO.puts "hello world"
                  private_function()
                end

                defp private_function do
                  IO.puts "I am private"
                end
              end
Write a string

              name = "This is a string"
Search a string

              String.contains?("is", "This is a string")
Replace a string

              String.contains?("This is a string", "is")
Substitute a string

              String.replace("This is a string", "is", "IS")
Split a string into an array

              String.split("This is a string", " ")
Do regex match
Below are the String and Regex solutions. Regex module contains a lot more functions and documentation on regex expressions in Elixir. 

              String.match?("This is a string", ~r/^This/)
              Regex.match?(~r/^This/, "This is a string")
Split a string into an array

              String.split("This is a string", " ")
Write a conditional
Elixir has three ways of writing conditionals: cond, case, if. 

              # case is for pattern matching
              case value do
                "apples"  -> apple_fuction()
                "bananas" -> banana_function()
                _         -> default()
              end

              # cond is for a series of conditionals.
              # Equivalent to lispy cond
              cond do
                x > 10 -> greather()
                x < 10 -> lesser ()
                true   -> equal()
              end

              # And we have if, for standard if then clauses
              if x == 10 do
                  "goody"
              else 
                  "baddy"
              end
